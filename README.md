# Mangalam HDPE Pipes

A static landing page for Mangalam HDPE Pipes featuring a product showcase, interactive image carousel, modal forms, FAQ accordion, and responsive layout.

## Features

- Interactive product image carousel with thumbnail navigation  
- Zoom lens effect for detailed product viewing  
- Sticky header and product info bar for improved user experience  
- Modal forms for catalogue requests and callback inquiries  
- Fully responsive design (mobile-first approach)  
- FAQ accordion with smooth toggle functionality  
- Tab-based product information section  

## Files

- `index.html` — main entry point
- `styles.css` — page styling
- `script.js` — interactive behavior and modal handling
- `images/` — logo and product assets

## Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid)  
- Vanilla JavaScript (ES6)  

## Run locally

1. Open `index.html` directly in your browser
2. Or use a simple local server:

```bash
cd "Gushwork assignment"
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages

1. Push the repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` → `Pages`.
4. Set the source to:
   - Branch: `main`
   - Folder: `/ (root)`
5. Save.

Your site will publish at:

```
https://<your-github-username>.github.io/<repo-name>/
```

## Notes

- This repo is a static website and does not require a build step.
- The forms currently show mock alerts and do not submit real data.
- Remove `.DS_Store` from tracking by adding it to `.gitignore` if needed.
